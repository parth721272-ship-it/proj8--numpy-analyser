# 📊 NumPy Analyzer

Welcome to **NumPy Analyzer** – your all-in-one toolkit for data analysis using Python and NumPy! 🚀

---

## 🎯 Objective

NumPy Analyzer is a menu-driven, interactive Python application that empowers users to perform common data operations, statistical analyses, and mathematical computations on datasets using NumPy arrays. Designed with Object-Oriented Programming (OOP) principles, it’s perfect for students, data enthusiasts, and professionals alike!

---

## 🛠️ Features

- **Class-Based Design:**  
  All functionalities are encapsulated in the [`DataAnalytics`](project-8.py) class for clean, modular code.

- **Array Management:**  
  - Create 1D, 2D, or 3D arrays with custom sizes.
  - Index and slice arrays to access specific elements, rows, columns, or subarrays.
  - Combine multiple arrays or split arrays into smaller parts.

- **Mathematical Operations:**  
  - Perform element-wise addition, subtraction, multiplication, and division.
  - Matrix multiplication and dot product for 2D arrays.

- **Search, Sort, & Filter:**  
  - Search for specific values in arrays.
  - Sort arrays in ascending or descending order.
  - Filter arrays based on user-defined conditions.

- **Aggregating & Statistical Functions:**  
  - Compute sum, mean, median, standard deviation, and variance.
  - Find minimum and maximum values.
  - Percentiles and correlation coefficients between arrays.

- **User-Friendly Interface:**  
  - Intuitive menu-driven console interaction.
  - Easy navigation and clear prompts for every operation.
